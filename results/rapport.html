Analyse des Résultats

Observation: Le modèle couleur avec 5 classes obtient 20% d'accuracy, significativement inférieur aux 100% du modèle B&W.

Facteurs contribuants:
  • 5 classes vs 3: Plus de classes = plus difficile
  • 768 vs 64 entrées: 12× plus de paramètres = besoin de plus d'entraînement
  • Nouvelles classes: Angry et Surprised peuvent être plus ambigues
  • Données couleur: Les distinctions par couleur seule peuvent être insuffisantes

Recommandations:
  • Augmenter le nombre d'epochs (200+ au lieu de 100)
  • Augmenter la taille du modèle (hidden layer plus grand)
  • Ajouter plus de données d'entraînement
  • Data augmentation (rotation, translation)

🔧 Détails d'Implémentation

Architecture Réseau

Input (d neurons) ↓ W1 @ x + b1 [Linear] ↓ ReLU [Activation] ↓ W2 @ a1 + b2 [Linear] ↓ Softmax [Activation + Normalization] ↓ Output (C probabilities)

Forward Pass

z1 = W1 @ x + b1 a1 = max(0, z1) # ReLU z2 = W2 @ a1 + b2 output = exp(z2) / sum(exp(z2)) # Softmax

Loss Function

L = -sum(y_true * log(output)) # Cross-Entropy

Backpropagation

# Gradient couche sortie (simplifié: Softmax + CrossEntropy) delta2 = output - y_true # Gradient couche cachée delta1 = (W2^T @ delta2) * (z1 > 0) # ReLU derivative # Gradients poids grad_W1 = x @ delta1^T grad_W2 = a1 @ delta2^T # Update W1 -= lr * grad_W1 W2 -= lr * grad_W2

📈 Métriques de Classification

Définitions

Accuracy: (TP + TN) / Total
Precision: TP / (TP + FP) - "Parmi les positifs prédits, combien correctes?"
Recall: TP / (TP + FN) - "Parmi les vrais positifs, combien détectés?"
F1-Score: 2 * (Precision × Recall) / (Precision + Recall)
Macro F1: Moyenne des F1 par classe
